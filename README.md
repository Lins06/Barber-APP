💈 BARBERPRO
Sistema Desktop de Gestão para Barbearias
🧠 Sobre o Projeto

O BarberPRO é uma aplicação desktop desenvolvida para modernizar a gestão de barbearias, substituindo o tradicional caderno por uma solução digital, rápida e intuitiva.

Com foco em experiência do usuário (UX) e um design elegante no estilo Dark Gold, o sistema oferece controle completo de agenda, equipe e financeiro em um só lugar.

🚀 Funcionalidades

📊 Dashboard Visual
Visão geral da barbearia em tempo real
Cards com status da equipe e atendimentos

📅 Agenda Inteligente
Organização de horários por data
Ordenação automática por proximidade

👨‍🔧 Gestão de Equipe
Cadastro de barbeiros
Gerenciamento completo da equipe

✂️ Controle de Serviços
Tabela de preços editável
Suporte a múltiplos serviços

💰 Módulo Financeiro
Controle de faturamento
Relatórios diários e mensais

💻 Como Executar o Projeto

🔹 1. Clonar o repositório
git clone https://github.com/Lins06/Barber-APP.git
cd Barber-APP

🔹 2. Instalar dependências
npm install

🔹 3. Rodar o projeto
npm start
